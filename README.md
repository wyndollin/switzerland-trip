# Switzerland · Anna & Boio · 4 Days

A four-day editorial trip plan for the Bernese Oberland — built around old cogwheel trains, turquoise alpine lakes, and Bern's UNESCO Old Town.

## What's inside

A single `index.html` with:

- Editorial paper-aesthetic design (Fraunces + Lora + JetBrains Mono)
- Custom hand-drawn SVG illustrations for each day (no external images needed)
- Embedded **Google Maps** route view + clickable stop cards (11 locations)
- Day-by-day timeline with prices in CHF
- Per-person budget table (CHF + EUR)
- Verified booking links (Swiss Travel Pass, Schynige Platte, Oeschinensee, hotels)
- Reading-tips section for the trip

## The route

| Day | Route | Sleep |
|-----|-------|-------|
| 1 | Zürich → Interlaken → Schynige Platte → Gündlischwand | Interlaken |
| 2 | Interlaken → Spiez → Kandersteg → Oeschinensee | Kandersteg |
| 3 | Kandersteg → Spiez → boat → Thun → Bern Old Town | Bern |
| 4 | Bern → Zürich | — |

## Deploy

This is a single static HTML file. Drop into any host:

- **Vercel** — already configured via `vercel.json`. Run `npx vercel deploy --prod` or import the GitHub repo at vercel.com/new.
- **Netlify** — drag-and-drop `index.html` at app.netlify.com/drop
- **GitHub Pages** — enable in repo Settings → Pages → main branch
